🥛 Neel — The Dairy Products

📱 A full-stack mobile application to browse, order, and track fresh dairy products — built using **Django REST Framework** and **React Native (Expo)**.

---

✨ Features

- 🛍️ Browse dairy products with details & pricing  
- 🛒 Add to cart & seamless checkout  
- 💳 Payment integration (Stripe & Razorpay)  
- 📦 Real-time order tracking  
- 🔐 User authentication (Login/Register)  
- 📱 Clean mobile-first UI

🛠️ Tech Stack

- **Frontend**: React Native (Expo)  
- **Backend**: Django, Django REST Framework  
- **Database**: SQLite  
- **Payments**: Stripe, Razorpay  

---

⚙️ Setup

Backend
cd dairy-backend
python -m venv .venv
. .venv/Scripts/Activate.ps1
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

Frontend
cd dairy-frontend
npm install
npm run start
